Welcome to the online repository for theoretical undertakings regarding my honours thesis

This project was completed in partial fulfilment of my BSc. (honours) degree in Biochemistry. The repository contains all python code used in the development of a suitable optimization algorithm for the model by Kouril et al. (available here). The repository also includes any additional technical information regarding the project which may be deemed useful by moderators, or otherwise interesting to external readers.

Contents

  • Monte Carlo_levenberg marquardt optimization.ipynb - The complete optimization algorithm for the model available in a user-friendly readable format.
  • Monte Carlo_levenberg marquardt optimization.py - A copy of the complete optimization algorithm for the model available as an executable python script
  • Algorithm pseudocode.md - The complete algorithm represented as pseudocode for readers not specifically familiar with the python programming language
